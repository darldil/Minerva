# Instrucciones para generar la base de datos.

Como requisito indispensable es necesario disponer de XAMPP con el módulo "Apache" y "MySQL" ejecutados.

En PHPMyAdmin se debe crear una base de datos llamada "minerva", el usuario de acceso debe ser "root" y contraseña vacía.
Para cargar la información mínima en la base de datos copia el contenido de "CreacionBaseDatos.sql" y ejecutalo en el apartado
"SQL" de PHPMyAdmin.

El usuario administrador predeterminado en minerva es "admin@minerva.com" con contraseña "admin123".
